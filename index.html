Introduction

HTML and CSS are the foundational technologies used to create and design webpages. HTML structures the content on a page, while CSS styles it, making the web more visually appealing and user-friendly.

What is HTML?

HTML is a markup language used to create the structure and content of web pages. It uses tags to define elements like headings, paragraphs, images, and links.

Example

<!DOCTYPE html> <html> <head> <title>My First Web Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first webpage.</p> </body> </html>
Basic HTML Structure

An HTML document typically starts with a doctype declaration, followed by the <html>, <head>, and <body> tags. The head contains metadata, and the body contains the visible content of the webpage.

<!DOCTYPE html> <!-- Declares the document type --> <html> <!-- Root element --> <head> <!-- Metadata like title, styles, etc. --> <title>Page Title</title> </head> <body> <!-- Visible content --> <h1>Main Heading</h1> <p>Paragraph of text.</p> </body> </html>
Common HTML Tags

Some commonly used HTML tags include <h1> to <h6> for headings, <p> for paragraphs, <a> for hyperlinks, <img> for images, and <div> or <span> for grouping content.

Tag Description
<h1> to <h6> Headings
<p> Paragraph
<a href=""> Anchor/Link
<img src=""> Image
<ul>, <ol>, <li> Lists
<div> Division/Container
<span> Inline Contatiner
<form> Form for user input
What is CSS?

CSS, or Cascading Style Sheets, is a language used to style HTML documents. It controls the layout, colors, fonts, and overall appearance of elements on a webpage.

  • Define styles like colors, fonts, spacing, and layout.
  • Can be applied inline, internally, or externally.
Types Of CSS

There are three types of CSS: Inline CSS (inside HTML elements), Internal CSS (within a <style> tag in the head), and External CSS (linked using a separate .css file).

  1. Inline CSS – Inside an HTML element
    2. <p style="color: red;">This is red text.</p>
  2. Internal CSS – Inside a <style> tag in the <head>
    3. <style>
    body { background-color: lightblue; }
    </style>
  3. External CSS – In a separate .css file
    4. <link rel="stylesheet" href="styles.css">
CSS Syntax and Selectors

CSS syntax consists of a selector and a declaration block. Selectors target HTML elements, and declarations define how the elements should be styled using property-value pairs.

selector { property: value; } Example:
h1 { color: blue; font-size: 24px; } Types Of Selectors:
Selector Description
* universal Selector
#id Selects element by ID
.class Selects elements by class
element Selects by tag name
element1, element2 Multiple selectors
element1 element2 Descendant selector
Box Model in CSS

The CSS box model describes how elements are rendered on a page, including the content, padding, border, and margin. Understanding this model is crucial for precise layout control.

Every HTML element is a box composed of:

  • Content – The actual content
  • Padding – Space around content
  • Border – Surrounds padding
  • Margin – Space outside the border
div { margin: 10px; padding: 20px; border: 2px solid black; }
Positioning and Layout

CSS provides various positioning methods like static, relative, absolute, fixed, and sticky. It also allows layout designs using Flexbox and Grid for arranging elements efficiently.

Position Values:
  • static(default)
  • relative
  • absolute
  • fixed
  • sticky
Flexbox Example: .container { display: flex; justify-content: center; align-items: center; }
Media Queries and Responsiveness

Media queries in CSS enable websites to adapt to different screen sizes and devices. They are essential for building responsive designs that offer a good user experience on all devices.

Used to make websites responsive for different screen sizes.
@media (max-width: 600px) { body { background-color: lightgray; } }
Example Project (HTML + CSS)

In this section, we present a basic web project that combines HTML and CSS to demonstrate the structure and styling of a simple webpage, highlighting real-world application of the concepts discussed.

<!-- index.html --> <!DOCTYPE html> <html> <head> <title>Simple Web Page</title> <link rel="stylesheet" href="style.css"> </head> <body> <h1>Welcome to My Site</h1> <p>This is a styled paragraph.</p> </body> </html> /* style.css */ body { font-family: Arial, sans-serif; background-color: #f0f0f0; color: #333; padding: 20px; } h1 { color: navy; }
